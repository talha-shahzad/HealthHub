To run the project
Follow the following steps

1. Open Visual Studio Code in the project Folder
2. Open Terminal from VS code using command ctrl + '`'
3. Open server folder using command "cd server"
4. Run command in terminal "Node index.js"
5. Open another instance of terminal 
6. Open client Folder by using "cd Client"
7. Run command npm start in client side terminal.

The website will be opened in web browser.

If it did not open it would as to change the local host fro 3000 in terminal
just Press y, It will start running.
